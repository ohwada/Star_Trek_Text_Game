# Star Trek Text Game

# Star Trek Text Game は、1970年代に流行ったコンピュータゲームです。
私は、大学の研究室にあったミニコンで遊びました。

今でも、根強い人気があるようで、ネットに色々情報があり、ブログにまとめました。
http://android.ohwada.jp/archives/7339

Androidに移植しています。
まだ未完成ですが、簡単な遊びは出来るようになったので。ソースコードを公開します。
